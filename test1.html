2、类的定义

class 类名(object):    #括号里写的是父类名字，由于所有的类都object类的子类，所以当一个自定义类无直接父类时，括号里写上object,不同于C++，Python没有所谓的继承方式
    ……  #方法体
    def funct(self)  #类中的函数，即类的方法默认第一个参数必须是self

  • 类的调用方式

    变量名 = 类名()
    变量名.所调用类的方法名(参数表)

  • 累可以起到模板的作用，因此可以在创建实例的时候把一些我们认为必须绑定的属性强制填写进去，通过“init”方法，例如文档开头的类。此时的调用形式：

    a=Student("******","######")

3、私有变量和私有方法

  • 私有变量和私有方法通过在变量名和方法名前面加上“__",两个段横杠来实现，一旦定义为私有变量和私有方法，便无法在类体外访问。

    class PubAndPri:
        pub = "这是共有变量"
        __pri="这是私有变量"
        def __init__(self):
            self.other = "公有变量也可以这样定义"
        def out_pub(self):
            print("公有方法",self.pub,self.__pri)
        def __out_pri(self):
            print("这是私有方法",self.pub,self.__pri)
    ​
    pp=PubAndPri()
    pp.out_pub()
    print(pp.pub,pp.other)
    try:
        pp.__out_pri()
    except Exception as e:
        print("调用私有方法错误！")

字符串操作

  • str.split可以将字符型转换成list

    >>> s = "I love python,and\nyou?\thehe"
    >>> print(s)
    I love python,and
    you?    hehe
    >>> s.split(",")
    ['I love python', 'and\nyou?\thehe']

  • 字符串检测开头和结尾，可以用：

    string.endswith('str')、string.startswith('str')
    file = "F:\\data\\catering_dish.xls"
    >>> file.endswith('xls')
    True
    >>> url = "https://www.i-nuc.com"
    >>> url.startswith('https')
    True

  • 要将字符串中的某个字符替换掉可以使用replace(被查找词，替换词)

    >>> s='I love python, do you love python too?'
    >>> s.replace('python','R')
    'I love R, do you love R too?'
    >>>s.replace('python','C++',1)
    'I love C++, do you love python too?'

  • re.sub(被替词，替换词，替换域，flags=re.IGNORECASE)这条语句用于查找与替换，忽略大小写

    >>> S = 'I love python,do you love Python'
    >>> re.sub('python','C++',S,flags=re.IGNORECASE)
    'I love C++,do you love C++'

  • 关于备份，当处理数据时需要备份数据时，不可以简单地直接用“a=b",应该用”b=a[:]"和copy（）。

    >>> a=[3,2,5,4,9,8,1]
    >>> id(a)
    49461408
    >>> c=a
    >>> id(c)
    49461408
    >>> b=a[:]
    >>> id(b)
    60696496
    >>> d=a.copy()
    >>> id(d)
    60696816

  • 关于下划线“—”。

    下划线对解释器有特殊含义，而且是内建标识符所使用的符号，故一般不应该用想下划线作变量名的开头，如_xxx会被看做“私有的”，\\ __xxx代表私有成员，而变量名前后都有双下划线多Python具有特殊含义。总而言之不可以随意使用下划线开头命名变量。
